About this project

TSyWeb Project

  • This project is a web platform designed to provide users with an interactive gaming experience and the ability to communicate through a real-time chat. The application is built using modern technologies such as Spring Boot for the backend and Angular for the frontend, with the inclusion of WebSockets to enable real-time communication.
  • Real-Time Chat: The platform offers users the ability to communicate with each other through a real-time chat. Users can send text messages, and enjoy an interactive conversation experience.
  • Online Games: The platform offers two exciting online games that users can enjoy. These games are accessible directly from the web browser, and users can compete against each other in real-time.
  • MySQL Database: User information, profiles, and game data are stored in a MySQL database to keep track of user activity and game progress.
