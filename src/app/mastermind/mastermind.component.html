How to play MasterMind
The game consists of guessing a randomly generated combination of 6 colors. Players will create a combination on each turn. The game will end when one of the players manages to guess the master combination or when the attempts run out (6 per player). The player can create combinations using the following colors:
The combination can have the same color repeated several times.
Successful or failed attempts regarding the master combination are reflected in small black, white, or empty balls placed to the right of the combination. The possibilities are:
- : Both the color and the position were guessed correctly.
- : Only the color was guessed, but it's in another position.
- : The color is not present in the master combination.
